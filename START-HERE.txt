BLACKBOX DESKTOP EXPERIENCE TEST BUILD

1. Extract the entire ZIP to a normal folder.
2. Double-click Blackbox.App.exe.
3. Use Capture > Setup OBS and wait for the ready message.

The first setup checks normal and Steam OBS locations first. If OBS is already
installed, Blackbox prepares its private runtime from those local files without a
download. Otherwise, it downloads and verifies the official OBS Studio package.
Administrator access is not required.

After setup succeeds:

1. Start a game and leave its main window running.
2. Open the Games drawer, click Manage remembered games, select the game under
   Running applications, and click Remember as game.
3. Select the remembered profile to control automatic recording, game audio,
   launcher handoff, GPU preference, and executable aliases.
4. Open Settings and enable Watch remembered games.

Blackbox remembers that executable for future runs. When the game starts, it
detects the live window and size, configures OBS game video and process audio,
then begins recording. Disable Watch remembered games to stop automatic
watching. You can pause or remove remembered games from the Games window.

Settings also includes Start with Windows and Close to notification area.
Windows startup launches Blackbox quietly in the notification area. The
notification icon can open Blackbox, start or stop recording, protect the
previous five minutes, toggle game watching, open recordings, or exit.

Recordings and the short setup test recording are stored in your
Videos\Blackbox folder. Click Recordings to play a continuous session or export
it as one MKV or MP4. The Blackbox player includes segment-marked scrubbing,
10-second and segment jumps, frame stepping, speed, loop, audio-track, volume,
fullscreen, quick tags, and manual event markers.

Timeline thumbnails and waveforms load from the preview cache when available.
For a recording without a cached preview, click Build preview when you want
Blackbox to generate those visual assets. Playback does not require a preview.

The private OBS runtime is separate from your normal or Steam OBS profile. Game
sources remain dormant until an enabled remembered game is running. Blackbox
can learn a launcher's final game executable after two matching scans, and the
Games window lets you review or remove every learned alias.

Logs: %LOCALAPPDATA%\Blackbox\logs

For troubleshooting, open Diagnostics and click Support bundle. Blackbox shows
the privacy contents before it creates the local ZIP. The bundle excludes all
recordings, screenshots, databases, OBS passwords and settings, microphone
configuration, saved game profiles, executable lists, and application settings.
