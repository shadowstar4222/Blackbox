1. Extract the entire ZIP to a normal folder.
2. Double-click Blackbox.App.exe.
3. Click Setup OBS and wait for the ready message.

The first setup checks normal and Steam OBS locations first. If OBS is already
installed, Blackbox prepares its private runtime from those local files without a
download. Otherwise, it downloads and verifies the official OBS Studio package.
Administrator access is not required.

After setup succeeds:

1. Start a game and leave its main window running.
2. Click Games, select the game under Running applications, and click
   Remember as game.
3. Select the remembered profile to control automatic recording, game audio,
   launcher handoff, GPU preference, and executable aliases.
4. Close the Games window and click Enable Auto.

Blackbox remembers that executable for future runs. When the game starts, it
detects the live window and size, configures OBS game video and process audio,
then begins recording. Disable Auto to stop the automatic recording. You can
pause or remove remembered games from the Games window.

Recordings and the short setup test recording are stored in your
Videos\Blackbox folder. Click Recordings to play a continuous session or export
it as one MKV or MP4.

The private OBS runtime is separate from your normal or Steam OBS profile. Game
sources remain dormant until an enabled remembered game is running. Blackbox
can learn a launcher's final game executable after two matching scans, and the
Games window lets you review or remove every learned alias.

Logs: %LOCALAPPDATA%\Blackbox\logs
